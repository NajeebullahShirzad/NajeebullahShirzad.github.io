Handwritten Digit Classification Using ML Models

Machine Learning | Deep Learning | Computer Vision

Project Overview

This project focuses on building and evaluating multiple machine learning and deep learning models to classify handwritten digits from the MNIST dataset. The dataset contains 70,000 grayscale images (28×28 pixels) representing digits from 0 to 9. The goal was to compare traditional ML models with neural network–based approaches and identify the most effective method for image classification.

Models Implemented

  • Support Vector Machine (SVM)
  • Random Forest Classifier
  • k-Nearest Neighbors (k-NN)
  • Deep Neural Network (DNN)
  • Convolutional Neural Network (CNN)

Key Contributions

  • Loaded and explored the MNIST dataset using Scikit-learn and Keras.
  • Normalized pixel values and reshaped input data for model compatibility.
  • Trained classical ML models with and without hyperparameter tuning using GridSearchCV.
  • Designed and trained DNN and CNN models using TensorFlow/Keras.
  • Evaluated models using accuracy, precision, recall, F1-score, and confusion matrices.

Results Summary

  • SVM: 96.30% test accuracy — best-performing traditional ML model.
  • DNN: 95.08% validation accuracy after 20 epochs.
  • CNN: 99.10% test accuracy — highest overall performance.
